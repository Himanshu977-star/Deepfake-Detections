# Deepfake Detection System

A machine learning/deep learning based project that detects whether an image/video is **real** or **deepfake/manipulated** using computer vision techniques.

---

## 📌 Project Overview

Deepfake technology uses artificial intelligence to create fake but realistic-looking media.  
This project aims to identify manipulated/fake facial images or videos using deep learning models trained on facial datasets.

The system analyzes input media and predicts whether it is:

- **Real**
- **Fake / Deepfake**

---

## 🚀 Features

- Upload image/video for detection
- Detect manipulated facial content
- Real-time prediction output
- User-friendly web interface using Gradio
- Confidence score for predictions
